# Dolu Tool

**Dolu Tool** is a powerful and **free** in-game development tool for FiveM.

Teleport anywhere, spawn and move objects with a gizmo, debug MLO interiors and portals, browse every vanilla ped, vehicle and weapon, control time and weather — all from a single in-game menu.
